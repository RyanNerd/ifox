iFox.SMTP
Introduction

iFox.SMTP allows you to send E-Mails from your application without the need of a mail program. Messages sent follow the guidelines stablished in RFC 2822, 2045, 2046, 2047, 2048 and 2049 allowing interoperatibility with all kinds of SMTP server. Using iFox.SMTP you can:
  • Send text or HTML messages
  • Insert images in HTML messages
  • Attach files
  • Use multiple receivers, including CCO
This component uses iFox.Sockets services to establish communication with the SMTP server, thus the same requirements indicated for this componente applies.
