iFox.Connect
Introduction

iFox.Connect goal is to make easier to manage Internet connections. Using iFox.Connect you can:

  • Connect to the Internet using any connection defined in Dial-up Networking
  • Disconnect from the Internet
  • Enumerate available connections
  • Determine if an active Internet connection already exists
