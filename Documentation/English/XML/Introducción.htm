iFox.XML
Introduction

iFox.XML allows you to incorporate XML manipulation to any application. Using iFox.XML it is possible:
  • To convert a cursor (or several ones) to XML format and back.
  • To dinamically generate an XML file.

Note: As Visual FoxPro cursors can't be transfered trough a COM interface, most of iFox.XML capabilities can be used just from iFox.VFPScript. To widen its usage area this object has been extracted and compiled separatedly in order to allow it to be used right from Microsoft Visual FoxPro. You can get this version from iFox's download area.
