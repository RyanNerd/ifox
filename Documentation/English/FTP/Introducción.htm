iFox.FTP
Introduction

iFox.FTP provides an implementation of the FTP protocol allowing every kind of application to interact with an FTP server. Using iFox.FTP you can:

  • Upload and download files
  • Create and delete directories
  • Get a directory content
  • Rename and delete files
This component uses the proven Microsoft WinInet library, so having installed Microsoft Internet Explorer 4 or higher is required.
