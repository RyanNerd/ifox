iFox.FTP
Samples

The following examples show different ways to use iFox.FTP


Sample 1: Transferring a file to an FTP server.

oFTP = CreateObject("iFox.FTP")
If oFTP.Connect("ftp.server.com", "user", "xxxx")
   oFTP.Upload("C:\DATA.ZIP", "/ARCHIVES/DATA.ZIP")
   oFTP.Close()
Endif


Sample 2: Download a file and monitor the transfer progress.

oFTP = CreateObject("iFox.FTP")
If oFTP.Connect("ftp.server.com", "user", "xxxx")

   If oFTP.Startdownload("/ARCHIVES/DATA.ZIP", "C:\DATA.ZIP")

      Do While !oFTP.EOT
         If oFTP.DownloadNextPart()
            ? "Bytes transferred: " + Str(oFTP.TransferredBytes)
         else
            ? "Error"
         Endif
      Enddo

   else
      ? "Error"
   Endif

   oFTP.Enddownload()

   oFTP.Close()

Endif
