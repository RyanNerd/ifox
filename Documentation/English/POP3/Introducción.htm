iFox.POP3
Introduction

iFox.POP3 allows receiving messages from any POP3 server. Using iFox.POP3 it is possible:

  • Receive messages in text or HTML format
  • Receive embedded images in HTML messages
  • Receive attachements
  • Download headers only
  • Delete messages without the need of download them

This component uses iFox.Sockets services to establish communication with the POP3 server, thus the same requirements indicated for this componente applies.
