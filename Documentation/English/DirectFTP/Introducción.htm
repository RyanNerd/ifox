iFox.DirectFTP
Introduction

This component performs the same operations than iFox.FTP but it was written from scratch by the iFox team and doesn't suffer from WinInet limitations. Using iFox.DirectFTP it is possible to resume interrupted downloads, to move files and to add information to an existing file.

This object uses iFox.Sockets services to establish communication with the FTP server, thus the requirements indicated for this component applies to this one as well.
