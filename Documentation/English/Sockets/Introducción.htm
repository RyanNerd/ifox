iFox.Sockets
Introduction

iFox.Sockets encapsulates Winsock 2 complexity in a simple-use component. Using iFox.Sockets it is possible:
  • Establish multiple simultaneous connections, even to several ports.
  • Create listening sockets and accept connections.
  • Determine the IP address corresponding to a host and viceversa
  • Perform a Ping
Requirements: Winsock 2 or higher installed (it is included with Windows 98 or higher). If you are using Windows 95 you should install Microsoft Internet Explorer or download an update from Microsoft web site.
