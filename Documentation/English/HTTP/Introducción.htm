iFox.HTTP
Introduction

iFox.HTTP provides access to the HTTP protocol, making easier to automate the execution of tasks usually made manually trough a web browser. Using iFox.HTTP it is possible:

  • Get the content of a web page
  • Simulate a web form
  • Upload a file programatically
