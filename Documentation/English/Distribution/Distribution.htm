iFox
Distribution

To include iFox in your application add the following files in your installer project:

ifox.dll
vfp8t.dll
vfp8renu.dll

Use only the files located in the BIN folder that you will find in iFox's installation folder. Instruct the installer program to register iFox.DLL.

Additionally you need the file msvcr70.dll and, in case you have installation problems, the file gdiplus.dll.

In this last cases you can use the files provided with iFox or, if you was already including them in your setup, those ones.
