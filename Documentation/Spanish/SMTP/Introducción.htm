iFox.SMTP
Introducción

iFox.SMTP permite enviar E-Mails directamente sin necesidad de ningún cliente de correo. Los mensajes enviados cumplen con los RFC 2822, 2045, 2046, 2047, 2048 y 2049 permitiendo la interoperabilidad con todo tipo de servidores SMTP. Utilizando iFox.SMTP es posible:
  • Enviar mensajes en formato de texto o HTML
  • Insertar imágenes en los mensajes HTML
  • Adjuntar archivos
  • Soporte de múltiples destinatarios, incluso ocultos
Este objeto utiliza los servicios de iFox.Sockets para establecer la comunicación con el servidor SMTP, por lo tanto se aplican los requerimientos indicados para este componente.
