iFox.HTTP
Introducción

iFox.HTTP provee acceso al protocolo HTTP, facilitando la ejecución automatizada de tareas que de lo contrario deberían realizarse manualmente utilizando un navegador. Utilizando iFox.HTTP es posible:
  • Obtener el contenido de una página web
  • Simular un formulario
  • Subir un archivo programáticamente
