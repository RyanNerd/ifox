iFox.POP3
Introducción

iFox.POP3 permite recibir mensajes de cualquier servidor POP3. Utilizando iFox.POP3 es posible:
  • Recibir mensajes en formato texto o HTML
  • Recibir imágenes insertadas en los mensajes HTML
  • Recibir archivos adjuntos
  • Descargar solamente los encabezados
  • Eliminar mensajes sin necesidad de descargarlos
Este objeto utiliza los servicios de iFox.Sockets para establecer la comunicación con el servidor POP3, por lo tanto se aplican los requerimientos indicados para este componente.
