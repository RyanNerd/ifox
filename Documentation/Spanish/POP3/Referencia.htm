Propiedades
Nombre Descripción
UserName Tipo de valor: String

Nombre de usuario. Este valor será utilizado para efectuar la autenticación con el servidor POP.
Password Tipo de valor: String

Clave de Acceso. Este valor será utilizado para efectuar la autenticación con el servidor POP.
MessageCount Tipo de valor: Integer

Cantidad de mensajes recibidos. Este valor solamente es válido luego de una llamada exitosa al método Connect.
TotalSize Tipo de valor: Integer

Tamaño total en bytes de los mensajes recibidos. Este valor solamente es válido luego de una llamada exitosa al método Connect.
DownloadHeaders Tipo de valor: Logical

Indica si al recibir un mensaje solamente debe descargarse el encabezado.
DownloadLines Tipo de valor: Integer
Valor por Defecto: 10

Si la propiedad DownloadHeader contiene el valor Verdadero ademas del encabezado se descargarán la cantidad de líneas del cuerpo indicadas por esta propiedad.
ErrorNumber Tipo de valor: Long

En caso de producirse un error esta propiedad contendrá el número de error ocurrido. Los valores posibles son:

1. Error de conexión
2. No se ha obtenido respuesta del servidor POP3
3. El servidor POP3 ha respondido con un mensaje de error
4. Al intentar analizar el mensaje se ha determinado que este es inválido.
12. Se ha rechazado el nombre de usuario o la clave de acceso.
POP3Response Tipo de valor: String

Contiene una descripción del error ocurrido. El valor contenido dependerá del código de error indicado por la propiedad ErrorNumber.

Si el código de error es 3 esta propiedad contendrá la respuesta recibida del servidor POP3. Para los otros códigos de error esta propiedad contendrá una cadena vacía.
RawHeaders Tipo de valor: String

Contiene el encabezado del mensaje sin procesar. Esta propiedad resulta útil si el usuario desea efectuar su propia rutina de decodificación.

Este valor solamente es válido luego de una llamada exitosa al método Get.
RawBody Tipo de valor: String

Contiene el cuerpo del mensaje sin procesar. Esta propiedad resulta útil si el usuario desea efectuar su propia rutina de decodificación.

Este valor solamente es válido luego de una llamada exitosa al método Get.
SentDate Tipo de valor: String

Contiene la fecha de envío del mensaje.

Este valor solamente es válido luego de una llamada exitosa al método Get.
SentTime Tipo de valor: String

Contiene la hora de envío del mensaje. La hora se encuentra expresada utilizando la zona horaria del remitente.

Este valor solamente es válido luego de una llamada exitosa al método Get.
SentTimeZone Tipo de valor: Integer

Contiene la zona horaria del remitente. Puede utilizar este valor para ajustar el valor indicado en la propiedad SentTime a su propia zona horaria.

Este valor solamente es válido luego de una llamada exitosa al método Get.


Métodos
Nombre Descripción
Connect Parámetros: cServer as String, nPort as Long

Respuesta: Logical

Establece una conexión con un servidor POP3.

El parámetro cServer debe contener el nombre o dirección IP de un servidor POP3. El parámetro nPort indica el número de puerto a utilizar, en caso de no ser especificado se utilizará el puerto estandar establecido en el protocolo POP3: 110.

Antes de liberar el objeto se debe cerrar la conexión utilizando el método Disconnect.

La respuesta obtenida indica si la conexión ha podido establecerse exitosamente.
Disconnect Parámetros: Ninguno

Respuesta: Logical

Cierra una conexión establecida utilizando el método Connect.

La respuesta obtenida indica si se ha producido un error al intentar cerrar la conexión.
Delete Parámetros: nMessage as Integer

Respuesta: Logical

Elimina el mensaje indicado por el parámetro nMessage. Este valor debe ser un número comprendido entre 1 y el indicado por la propiedad MessageCount.

De acuerdo a las reglas del protocolo POP3 el mensaje no es eliminado realmente hasta que se cierra la conexión mediante el método Disconnect. Hasta ese momento puede ser recuperado llamando al método Recall. Luego de eliminado un mensaje se convierte en inválido y cualquier intento de descargarlo solamente obtendrá un mensaje de error del servidor POP3.

La respuesta obtenida indica si se ha producido un error al intentar ejecutar la operación.
Recall Parámetros: Ninguno

Respuesta: Logical

Recupera los mensajes eliminados durante la sesión activa.

Luego de cerrada la conexión mediante el método Disconnect los mensajes serán eliminados definitivamente sin existir forma de recuperarlos.

De acuerdo a las reglas del protocolo POP3 TODOS los mensajes borrados son recuperados.

La respuesta obtenida indica si se ha producido un error al intentar ejecutar la operación.
Test Parámetros: Ninguno

Respuesta: Logical

Este método envía un mensaje de prueba al servidor con el fin de verificar que la comunicación no se ha interrumpido o que el servidor ha dejado de responder.

La respuesta obtenida indica si se ha producido un error al intentar ejecutar la operación.
Get Parámetros: nMessage as Integer

Respuesta: Logical

Descarga el mensaje indicado por el parámetro nMessage. Este valor debe ser un número comprendido entre 1 y el indicado por la propiedad MessageCount.

Si la propiedad DownloadHeaders es Verdadera solamente se descarga el encabezado del mensaje y las líneas del cuerpo indicadas por la propiedad DownloadLines.

La respuesta obtenida indica si se ha producido un error al intentar ejecutar la operación.

Al finalizar la descarga las siguientes propiedades contendrán la siguiente información sobre el mensaje:


RawMessage
El mensaje "en bruto". Utilice esta propiedad para guardar el mensaje en formato EML o para hacer su propia decodificación.


RawHeaders
Los encabezados del mensaje "en bruto".


RawBody
El cuerpo del mensaje "en bruto".


SenderName
El nombre del remitente del mensaje.


SenderEMail
El EMail del remitente del mensaje.


RecipientsCount
La cantidad de destinatarios del mensaje. Si se ha utilizado CCO esta cantidad puede ser cero.


Recipients
Una colección con los nombres y direcciones de EMail de los destinatarios. Leer de la siguiente manera: 
For x = 1 to oPOP3.RecipientsCount
    cName = oPOP3.Recipients[x].ReceiverName
    cEMail = oPOP3.Recipients[x].ReceiverEMail
Next
CCCount
La cantidad de destinatarios con copia del mensaje. Si se ha utilizado CCO esta cantidad puede ser cero.


CC
Una colección con los nombres y direcciones de EMail de los destinatarios con copia. Leer de la siguiente manera: 
For x = 1 to oPOP3.RecipientsCount
    cName = oPOP3.Recipients[x].ReceiverName
    cEMail = oPOP3.Recipients[x].ReceiverEMail
Next
Subject
El asunto del mensaje.


Priority
La prioridad del mensaje. Los valores posibles son:
1. Alta
3. Normal
5. Baja


SentDate
La fecha de envío del mensaje indicada utilizando la zona horaria del remitente.


SentTime
La hora de envío del mensaje indicada utilizando la zona horaria del remitente.


SentTimeZone
La zona horaria del remitente. Puede utilizar esta información para ajustar la hora de envío a cualquier zona horaria.


AdjustedDate
La fecha de envío del mensaje expresada en la zona horaria indicada en el Panel de Control de Windows.


AdjustedTime
La hora de envío del mensaje expresada en la zona horaria indicada en el Panel de Control de Windows.


Body
El cuerpo del mensaje en formato de texto plano. Solamente disponible si el programa remitente ha codificado el mensaje en este formato.


HTMLBody
El cuerpo del mensaje en formato HTML. Solamente disponible si el programa remitente ha codificado el mensaje en este formato.


AttachmentsCount
La cantidad de archivos adjuntos al mensaje.


Attachments
Una colección conteniendo los archivos adjuntos.
Leer de la siguiente manera: 
For x = 1 to oPOP3.AttachmentsCount
    cName = oPOP3.Attachments[x].FileName
    nSize = oPOP3.Attachments[x].FileSize
Next

ResourcesCount
La cantidad de archivos embebidos en el mensaje, por ejemplo: imágenes o sonidos.


Resources
Una colección conteniendo los archivos embebidos.
Leer de la siguiente manera: 
For x = 1 to oPOP3.ResourcesCount
    cName = oPOP3.Resources[x].FileName
    cContent = oPOP3.Resources[x].Content
    cID = oPOP3.Resources[x].ID
Next
ParseMessage Parámetros: cMessage as String

Respuesta: Logical

Permite decodificar un mensaje almacenado previamente pasando su contenido como parámetro. La respuesta obtenida indicará si se ha podido interpretarlo correctamente. Se establecen todas las propiedades indicadas en la documentación del método Get.
