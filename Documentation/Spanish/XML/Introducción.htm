iFox.XML
Introducción

iFox.XML permite incorporar el manejo de XML a cualquier aplicación. Utilizando iFox.XML es posible:
  • Convertir un cursor (o varios) a formato XML y viceversa
  • Generar un archivo XML dinámicamente
Nota: Debido a que los cursores de Visual FoxPro no pueden transferirse a través de la interfase COM, la mayoría de las facilidades de iFox.XML solamente pueden aprovecharse desde iFox.VFPScript. Para ampliar su espectro de utilización este objeto ha sido extraido y compilado separadamente de tal forma que pueda ser utilizado directamente desde Microsoft Visual FoxPro. Puede obtener esta versión en el área de descargas de iFox.
