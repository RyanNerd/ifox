iFox.Download
Introducción

iFox.Download permite generar cualquier tipo de archivo dinámicamente para ser utilizado en una página web o descargado. Utilizando iFox.Download es posible:
  • Mostrar una imagen almacenada en una base de datos
  • Generar en forma dinámica un archivo para descargar
