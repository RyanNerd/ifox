iFox.Download
Referencia

Propiedades
Nombre Descripción
Type Tipo de valor: Integer
Valor por Defecto: DOWNLOAD_STRING

Indica la forma en que se proporciona la secuencia de bytes a devolver. Los valores permitidos son:

DOWNLOAD_STRING: Se debe establecer la propiedad Data con la secuencia de bytes a devolver.

DOWNLOAD_FILE: Se debe establecer la propiedad Data con el nombre de un archivo que contenga la secuencia de bytes a devolver.
FileName Tipo de valor: String

Indica el nombre que se asignará al archivo "virtual" generado. En caso de utilizar el iFox.Download para generar un archivo a descargar este es el nombre que se mostrará al usuario. Si la propiedad Type contiene el valor DOWNLOAD_FILE y no se ha asignado un valor a esta propiedad se utilizará el nombre del archivo físico empleado.
Data Tipo de valor: String

Contiene la secuencia de bytes a devolver. Si la propiedad Type contiene el valor DOWNLOAD_FILE esta propiedad debe establecerse con el nombre de un archivo que contenga la secuencia de bytes a devolver.


Métodos
Nombre Descripción
Send Parámetros: Ninguno

Respuesta: Ninguna

Envía la secuencia de bytes indicada en la propiedad Data simulando que se trata de un archivo físico. De acuerdo a la forma en que iFox.Download sea utilizado el archivo generado puede ser utilizado dentro de una página web o descargado por el usuario.

Nota: Una página ASP que utilice iFox.Download no debe generar ningun otro tipo de respuesta a traves del método Response.Write o por medio de código HTML incrustado en la página ya que esta respuesta sería agregada al archivo generado. Los encabezados HTTP también son alterados, por lo tanto en caso de haber enviado alguna respuesta antes de llamar a este método se producirá un error en tiempo de ejecución.
