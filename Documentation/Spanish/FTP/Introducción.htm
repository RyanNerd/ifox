iFox.FTP
Introducción

iFox.FTP provee una implementación del protocolo FTP permitiendo a todo tipo de aplicaciones interactuar con un servidor FTP. Utilizando iFox.FTP es posible:
  • Subir y bajar archivos
  • Crear y eliminar directorios
  • Obtener el contenido de un directorio
  • Renombrar y eliminar archivos
Este componente utiliza la probada librería WinInet de Microsoft por lo que se requiere tener instalado Microsoft Internet Explorer 4 o superior.
