iFox.Resume&Go
Introducción

iFox.Resume&Go permite iniciar una descarga y continuarla en otro momento, al estilo de los más populares administradores de descargas. iFox.Resume&Go solamente soporta el protocolo HTTP, en caso de necesitar utilizar el protocolo FTP la misma funcionalidad se incluye en iFox.DirectFTP.

Este objeto utiliza los servicios de iFox.Sockets para establecer la comunicación con el servidor HTTP, por lo tanto se aplican los requerimientos indicados para este componente.
