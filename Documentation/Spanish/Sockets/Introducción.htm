iFox.Sockets
Introducción

iFox.Sockets encapsula la complejidad de Winsock 2 en un objeto simple de utilizar. Utilizando iFox.Sockets es posible:
  • Establecer múltiples conexiones simultáneas incluso a varios puertos.
  • Crear sockets de escucha y aceptar conexiones.
  • Determinar la dirección IP correspondiente a un nombre de host y viceversa
  • Efectuar un Ping
Requerimientos: Winsock 2 o superior instalado (se incluye con Windows 98 o superior). En caso de utilizar Windows 95 se debe instalar Microsoft Internet Explorer u obtener una actualización independiente de la página web de Microsoft.
