iFox.Connect
Referencia

Propiedades
Nombre Descripción
Connections Tipo de valor: Colección

Luego de una llamada exitosa al método EnumConnections este método contendrá una colección de objetos Connection representando las conexiones activas
ForceConnection Tipo de valor: Lógico

Si el valor de esta propiedad es verdadero el método Dial establecerá la conexión automáticamente sin pedir confirmación al usuario.


Métodos
Nombre Descripción
Dial Parámetros: cConnection as String (opcional)

Respuesta: Long

Establece una conexión a Internet. Puede solicitarse una conexión específica a través del parámetro cConnection. Si cConnection no es especificado o no corresponde a una conexión válida se utilizará la conexión por defecto.

Si la conexión no puede realizarse la respuesta será cero, caso contrario la respuesta es un handle identificatorio de la conexión que puede utilizarse para terminar la conexión a través del método HangUp. En caso de utilizar la conexión por defecto el valor devuelto es siempre -1.
EnumConnections Parámetros: Ninguno

Respuesta: Integer

Devuelve como respuesta la cantidad de conexiones activas.

Si no existiera ninguna conexión activa el método devolverá cero.

Si el valor devuelto es mayor o igual a uno la propiedad Connections contendrá una colección de objetos Connection representando las conexiones activas.
EnumEntries Parámetros: Ninguno

Respuesta: String

Devuelve como respuesta todas las conexiones disponibles (aquellas definidas en el Acceso Telefónico a Redes).

Si no existiera ninguna conexión disponible el método devolverá una cadena vacía.

Si existiera más de una conexión disponible se utilizará como separador los caracteres de Nueva Línea Chr(13) + Chr(10)
HangUp Parámetros: nConnection as Long (opcional)

Respuesta: Logical

Cierra una conexión activa a Internet. nConnection indica el handle correspondiente a la conexión que se desea terminar. Si no es especificado se cerrará la conexión por defecto.
IsConnected Parámetros: Ninguno

Respuesta: Logical

Determina si existe una conexión activa a Internet.
Statistics Parámetros: nConnection as Long, nType as Integer

Respuesta: Long

Permite obtener estadísticas sobre una conexión activa.

El parámetro nConnection debe ser el handle de una conexión, obtenido a través de los métodos Dial o EnumConnections. En Windows 95/98/Me este parámetro es ignorado ya que estos sistemas operativos no llevan estadísticas discriminadas por conexión.

El parámetro nType indica el tipo de estadística a obtener. Los valores que puede tomar este parámetro son:

1. Velocidad de conexión (en bps)
2. Bytes recibidos
3. Bytes enviados
4. Tiempo de conexión (en milisegundos). Bajo Windows 95/98/Me el valor devuelto en este caso siempre es cero debido a limitaciones del sistema operativo.
